# Finding Your Game's Executable

Installation requirements vary by user, platform, and game. As a result, games can be installed in different locations, making it difficult to locate the executable and corresponding game directory. This document describes a consistent method for locating a running game's executable file using Task Manager and File Explorer.

---

## Open Task Manager

Right-click the Windows taskbar and select **Task Manager**, or press **Ctrl + Shift + Esc**.

---

## Find the Game in Task Manager

In Task Manager, switch to the **Processes** tab, locate the running game process, then right-click it and select **Open file location**.

![Task Manager](https://assets.martysmods.com/additionalguides/gamelocation/FindGameTaskManager.webp)

A File Explorer window opens, highlighting the game's executable file in its installation directory. This directory is the location to use when guides refer to the "game folder" or "game directory."

![File Explorer](https://assets.martysmods.com/additionalguides/gamelocation/FindGameFileExplorer.webp)
